\subsection{Coding Styles}

\begin{itemize}
 \setlength{\parskip}{0cm}
 \setlength{\itemsep}{0cm}
 \item 移植性はできる限り高く
 \item マクロは基本大文字にする
 \item プログラム中に参考文献の論文を書く
 \item \verb|a != NULL -> a|, \verb|a == NULL -> !a|
 \item 改行する場合，演算子(\verb|+-*/|等)は行の先頭に書く
 \item Doxygenで変数は@a，予約語は@cを利用
 \item グローバル変数は大文字から始める
\end{itemize}
